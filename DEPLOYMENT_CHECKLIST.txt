================================================================================
COMMON ISSUES & SOLUTIONS
================================================================================

Issue: Sensor not connecting
--------------------------------------------------------------------------------
Solution:
1. Check USB connection
2. Verify serial port in r307.py
3. Check device permissions (Linux):
   sudo usermod -a -G dialout $USER


Issue: Import errors after deployment
--------------------------------------------------------------------------------
Solution:
1. Activate virtual environment
2. Run: pip install -r requirements.txt


Issue: 404 errors
--------------------------------------------------------------------------------
Solution:
1. Check URLs in urls.py
2. Verify DEBUG setting
3. Check ALLOWED_HOSTS


Issue: Database locked
--------------------------------------------------------------------------------
Solution:
1. Close all connections
2. Restart server
3. Consider using PostgreSQL for production
